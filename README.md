# english-spanish-learning-app
Final project - Stanford Code in Place 2026
## Description
This Python application helps users practice English and Spanish vocabulary through an interactive quiz.

## Features
-Spanish to English mode
-English to Spanish mode
-Random vocabulary questions
-Score tracking
-Vocabulary loaded from Excel

## Technologies Used
-Python
-Pandas
-Excel (.xlsx)
-GitHub
